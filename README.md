### Main Branch

The main branch will contain the in-development code. It is the default branch for the repository and will be used for ongoing development work. All new features, bug fixes, and improvements will be committed to this branch until they are ready for release.

When ready for release, they are being sent to the pre-release branch for testing and final adjustments before being merged into the release branch. This workflow allows for a structured development process while ensuring that the main branch remains up-to-date with the latest changes.

The website where people will be able to download the latest version of the software will be on the website branch
