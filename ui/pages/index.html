Backend Access List

Action / File Layer Parameters Description
ping server/api.py Server data: string Bridge health check. Echoes the payload back from Python through the C++ layer.
public_demo public/utils.py Public name: string Calls a public utility module. Intended for formatting helpers, non-sensitive API calls, and generic logic safe to expose.
private_demo private/secret_processor.py Private secret_data: string Calls a private module. Intended for databases, file I/O, encryption, and any sensitive operations hidden from the frontend.
modal_alert public/modals.py Public title: string message: string Shows a native OS warning dialog with an OK button. Blocks the Python thread until dismissed.
modal_info public/modals.py Public title: string message: string Shows a native OS information dialog with an OK button.
modal_error public/modals.py Public title: string message: string Shows a native OS error dialog with an OK button.
modal_confirm public/modals.py Public title: string message: string Shows a native OS Yes/No dialog. Returns true or false.
modal_prompt public/modals.py Public title: string message: string default?: string Shows a native OS text input dialog. Returns the entered string, or null if cancelled.

Native Bindings

Binding Layer Parameters Description
windowClose() Native — Posts WM_CLOSE to the native window handle.
windowMinimize() Native — Minimizes the window to the taskbar.
windowMaximize() Native — Toggles between maximized and restored states.
openModal(name) Native name: string Opens ui/modals/<name>.html as a new native OS window. Returns a Promise that resolves when the modal calls closeModal(value).
closeModal(value) Native value: any Closes the modal OS window and resolves the parent's Promise with value. Only available inside a modal's <script>.
dragWindow() Native — Initiates a native window drag. Bind to onmousedown on any drag region.
openExternalLink(url) Native url: string Opens a URL in the system browser. All <a href> clicks are intercepted automatically.
